# Upload These Files To GitHub Pages

Open this folder:

`github-pages-upload`

Everything inside this folder must go into the top level of the GitHub repository.

The final repository should look like this:

```text
rogersholdingsllc.github.io
├── index.html
├── styles.css
├── .nojekyll
└── assets
    └── images
        ├── eastland-logo.webp
        ├── welcome-pastors-slide.webp
        ├── sunday-services-slide.webp
        └── ...
```

Do not put the `github-pages-upload` folder itself into GitHub.

Upload the contents of the folder.
