# JPG-Only GitHub Pages Upload

Upload the contents of this folder to the top level of the GitHub repository.

The repository should look like this:

```text
index.html
styles.css
assets/
  images/
    eastland-logo.jpg
    welcome-pastors-slide.jpg
    sunday-services-slide.jpg
    ...
```

Do not upload this `github-pages-jpg-only` folder itself. Upload what is inside it.

This version uses JPG files only. There are no WebP image files in this folder.
